> 创建时间：2026-07-16
> 目标：Java 后端 + AI 应用开发简历项目

## 一、项目定位

基于 Spring Boot 4.0 + Spring AI 2.0 + React 18 + PostgreSQL + pgvector 的 AI 驱动非暴力沟通学习平台。作为简历项目，面向 Java 后端开发 / AI 应用开发岗位，需要同时体现**后端架构能力**和 **AI 应用开发能力**。

## 四、两层 Agent 架构（核心设计决策）

### 4.1 主 Agent（全能对话入口）

用户直接对话的智能助手，能调用所有模块工具。

```
用户: 帮我看看最近的练习情况
主Agent: [调用 dashboard_query] → "你最近一周练习了3次..."

用户: 生成一个和领导沟通的场景
主Agent: [调用 scenario_generate] → "已生成场景：..."

用户: 帮我记录一下今天的学习感悟
主Agent: [调用 wiki_write] → "已记录到你的个人 Wiki"
```

### 4.2 练习 Agent（多 Agent 编排）

专注于练习对话流程，10 种专职 Agent 通过 ModeRouter 策略路由。

```
场景驱动模式：
  CREATED → SCENARIO_GENERATOR（引入场景）
  后续 → DIFFICULT_PARTNER（扮演对方）
  评估触发 → NVC_EXPRESSION_EVALUATOR

结构化四步模式：
  OBSERVE → STEP_OBSERVE_COACH
  FEELING → STEP_FEELING_COACH
  NEED → STEP_NEED_COACH
  REQUEST → STEP_REQUEST_COACH
```

### 4.3 分工原则

| 维度 | 主 Agent | 练习 Agent |
|------|---------|-----------|
| 定位 | 全能对话入口 | 练习对话流程 |
| 工具调用 | 完整（10+ 工具） | 可选（RAG、评估 Skill） |
| 思考链 | CoT 引擎 | 无（直接调 LLM） |
| 多 Agent | 单 Agent | 多 Agent 编排 |
| 用户交互 | 自然语言完成所有操作 | 角色扮演 + 教练引导 |
