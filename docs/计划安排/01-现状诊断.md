# NVC 练习平台 — 现状诊断

> 排查时间：2026-07-16

---

## 一、当前已实现模块

### 1.1 练习模块（nvcpractice）— 核心模块

**已实现：**
- 3 种练习模式：SCENARIO（场景驱动）、FREE_DIALOG（自由对话）、STRUCTURED_FOUR_STEP（结构化四步）
- Agent 调度：NvcAgentOrchestrator + ModeRouter 策略模式
- 10 种 Agent 场景（NvcAgentScene 枚举）
- 对话服务：NvcAgentChatService（直接调 LLM，无工具）
- 评估服务：NvcEvaluationService（5 维度：观察/感受/需求/请求/共情）
- 摘要服务：NvcSummaryService（四要素提取，异步）
- 结构化练习：NvcStructuredPracticeService（四步推进）
- Agent 配置热更新（DB + Redis 缓存，5 分钟 TTL）
- 流式 SSE 对话 + 非流式对话
- Agent 配置管理页面（前端）

**缺失：**
- Agent 无工具调用能力（裸调 LLM）
- Agent 无思考链（CoT）
- Agent 无反思机制
- RAG 未集成到对话中
- 用户档案未注入上下文
- 评估未标准化为 Skill

### 1.2 场景库模块（nvcscenario）

**已实现：**
- 场景 CRUD（标题/描述/类型/难度/标签）
- AI 生成场景（结构化 JSON 输出）
- 5 种场景类型：WORKPLACE/FAMILY/INTIMATE/SOCIAL/SELF
- 13 个种子场景
- 使用次数统计

### 1.3 用户档案模块（nvcprofile）

**已实现：**
- 档案 CRUD（沟通背景/性格特征/沟通风格/情绪触发器）
- 能力分数管理（观察/感受/需求/请求/共情，5 维度）
- 能力雷达图（最近 10 次平均）
- 能力趋势图（最近 30 次）
- 沟通记录分析（AI 分析真实沟通场景）
- NVC 等级自动计算（BEGINNER/INTERMEDIATE/ADVANCED）
- 仪表盘统计

**缺失：**
- 档案更新手动（练习后不自动更新）
- 能力分数只有粗分（无子维度）
- 无基于档案的场景推荐

### 1.4 知识库模块（knowledgebase）

**已实现：**
- 文档上传（PDF/DOCX/TXT/MD，Apache Tika 解析）
- 文档分块（TokenTextSplitter，800 tokens/chunk）
- 向量化（DashScope text-embedding-v3，1024 维）
- RAG 查询（向量检索 + 查询重写 + 动态参数）
- 异步向量化管线（Redis Stream）
- 知识库 CRUD + 统计

**缺失：**
- 未与 NVC 练习对话集成
- 无多知识库分类管理
- 无个性化检索

### 1.5 语音模块（voiceinterview）

**已实现：**
- WebSocket 语音练习
- ASR（Qwen 语音识别）
- TTS（Qwen 语音合成）
- 语音会话管理
- 语音评估

**问题：**
- 模块名还是旧的 `voiceinterview`，未改为 `nvcvoice`

### 1.6 LLM Provider 模块（llmprovider）

**已实现：**
- 多 Provider 管理（DashScope/LmStudio/Kimi/DeepSeek/GLM/MiMo）
- API Key 加密存储
- ChatClient 缓存（ConcurrentHashMap）
- EmbeddingModel 管理
- 结构化输出（BeanOutputConverter + 重试 + 修复）

### 1.7 前端页面（9 个 NVC 页面 + 7 个组件）

| 页面 | 功能 |
|------|------|
| NvcPracticeHubPage | 练习中心（模式选择 + 难度选择） |

| NvcVoicePage | 语音练习（WebSocket + 实时字幕） |
| NvcHistoryPage | 练习历史列表 |
| NvcReportPage | 练习报告（评估详情 + PDF 导出） |
| NvcProfilePage | 个人档案（编辑 + 沟通分析） |
| NvcDashboardPage | 仪表盘（雷达图 + 趋势图） |
| NvcScenarioLibraryPage | 场景库（筛选 + AI 生成） |
| NvcAgentConfigPage | Agent 配置（Prompt 编辑 + 参数调节） |
