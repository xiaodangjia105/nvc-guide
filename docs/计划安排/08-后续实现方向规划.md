### 2.4 Agent 智能化（6 天）P0

#### 2.4.1 当前状态

Agent 框架已就绪（NvcTool 接口 + NvcToolRegistry + 10 个工具 + ModeRouter），但 Agent 的"智能"层面仍有较大提升空间：

- **无思考链（CoT）**：Agent 直接调用 LLM，没有显式的推理过程
- **无反思机制**：练习结束后不会回顾和分析对话质量
- **无自适应难度**：难度选择是手动的，不会根据用户表现自动调整
- **评估与对话脱节**：评估是独立调用，不是对话的自然组成部分
- **Agent 场景切换无记忆**：每次对话独立，不记住之前的交互
