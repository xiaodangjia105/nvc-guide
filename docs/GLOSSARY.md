# 领域术语表

> NVC 非暴力沟通练习助手的核心术语。新增术语时补入此文件。

## NVC 核心概念

| 术语 | 说明 |
|------|------|
| NVC | Nonviolent Communication，非暴力沟通。马歇尔·卢森堡提出的沟通方法论 |
| 四要素 | 观察（Observation）、感受（Feeling）、需求（Need）、请求（Request） |
| 观察 | 客观描述事实，不掺杂评判。"你迟到了 10 分钟"而非"你总是迟到" |
| 感受 | 识别和表达真实情绪，区分感受与想法。"我感到焦虑"而非"我觉得你不重视" |
| 需求 | 感受背后的普世人类需要。"我需要被尊重"而非"你需要改" |
| 请求 | 具体、可执行、可拒绝的行动请求。"你愿意提前 10 分钟出发吗"而非"你以后别迟到了" |
| 共情倾听 | 不带评判地倾听对方，尝试理解其观察/感受/需求/请求 |
| 长颈鹿 | NVC 中的象征，代表用心连接的沟通方式 |
| 杰克豺 | NVC 中的象征，代表评判、指责的沟通方式 |

## 系统术语

| 术语 | 说明 |
|------|------|
| Agent | NVC 专职 AI 角色，负责特定练习环节（场景生成、对话引导、评估等） |
| Agent 场景（Scene） | Agent 的职责分类，通过枚举定义，如 SCENARIO_GENERATOR、DIALOGUE_GUIDE |
| 结构化四步练习 | 按观察→感受→需求→请求四步逐一练习的模式 |
| 场景驱动练习 | 系统提供 NVC 场景，用户在场景中练习 |
| 自由对话练习 | 用户自由输入，Agent 实时引导和评估 |
| NVC 表达评估 | 对用户表达进行四要素完整性、准确性的评分 |
| 能力画像 | 用户在四要素维度上的能力雷达图 |
| 困难搭档（Difficult Partner） | 模拟 NVC 对话中难以沟通的对象 |
| 向量化个性化 | 基于 pgvector 的用户画像向量搜索，实现个性化场景推荐 |
