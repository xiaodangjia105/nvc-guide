# 系统约束

> 影响架构决策的关键约束条件。

## 技术约束
- **Java 21**：使用虚拟线程，不使用传统线程池
- **Spring Boot 4.0 + Spring AI 2.0**：AI 服务调用必须通过 LlmProviderRegistry
- **PostgreSQL + pgvector**：向量搜索 1024 维 COSINE，所有业务数据统一存储
- **Redis**：缓存、限流、异步任务（Stream），不可用其他 MQ 替代
- **单体仓库**：前后端同仓库，API 契约在仓库内对齐

## 业务约束
- **NVC 专业性**：Agent 回复必须符合 NVC 理论，四要素评估标准由领域专家定义
- **多语言**：当前仅中文，未来可能扩展英文
- **流式体验**：AI 对话必须流式输出，用户等待不超过 2 秒看到首个 token
- **评估实时性**：用户发消息后异步评估，评估结果延迟可接受（秒级）

## 安全约束
- API Key、数据库密码等敏感信息只在 `.env`，不入版本控制
- LLM 调用不记录用户原始对话到日志（隐私保护）
- 文件上传必须校验类型和大小
