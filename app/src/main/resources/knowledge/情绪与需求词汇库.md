## 三、需求与策略的区分

**核心原则：** 需求是普遍的、抽象的；策略是具体的、个人的。

| 需求 | 可能的策略（因人而异） |
|------|---------------------|
| 陪伴 | 每天一起吃饭 / 每周约会一次 / 每天聊天 30 分钟 |
| 尊重 | 进门前敲门 / 不在公共场合批评 / 征求意见后再决定 |
| 安全 | 每天报平安 / 不翻看手机 / 有事提前商量 |
| 自主 | 自己决定穿什么 / 可以有自己的爱好 / 不被催促 |
| 成长 | 报名培训班 / 得到建设性反馈 / 有挑战性的工作 |
| 理解 | 对方先听完再回应 / 不急着给建议 / 承认感受是合理的 |

**练习：** 当你发现自己在坚持某个"要求"时，问自己：
1. 如果这个要求满足了，我会感到什么？→ 找到感受
2. 这种感受说明我需要什么？→ 找到需求
3. 除了这个特定的方式，还有什么其他方式能满足这个需求？→ 打开策略空间
