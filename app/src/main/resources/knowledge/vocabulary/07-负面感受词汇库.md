# 负面感受词汇库

## 概述

在非暴力沟通中，准确识别和表达负面感受是自我理解和寻求支持的关键一步。许多人在表达负面情绪时容易混淆"感受"与"想法"，或将指责伪装成感受。本文档收录了 50 余个负面感受词汇，帮助你更清晰地觉察和表达内心的真实状态。

---

## 一、恐惧类感受

这类感受源于对安全、确定性的需要未被满足，常与未知或潜在威胁相关。

| 词汇 | NVC 表达示例 |
|------|-------------|
| 担心 | "你很晚还没回来，我感到担心。" |
| 害怕 | "听到雷声，孩子感到害怕。" |
| 焦虑 | "截止日期快到了，我感到焦虑。" |
| 不安 | "没有收到回复，我心里不安。" |
| 恐慌 | "体检结果异常，我感到一阵恐慌。" |
| 紧张 | "明天要上台演讲，我感到紧张。" |
| 忐忑 | "等待面试结果时，我心里忐忑不安。" |
| 提心吊胆 | "孩子一个人出远门，我提心吊胆的。" |
| 忧虑 | "看到父母年纪大了，我感到忧虑。" |
| 惶恐 | "突然被叫去谈话，我心里有些惶恐。" |
| 畏惧 | "面对未知的挑战，我感到畏惧。" |
| 发慌 | "钱包找不到了，我一下子慌了。" |
| 无所适从 | "面对两个都不错的选择，我感到无所适从。" |
| 心神不宁 | "有件事一直没处理完，我心神不宁。" |

| 不满 | "对这次的处理结果，我感到不满。" |

| 烦躁 | "被反复打断，我觉得很烦躁。" |
| 窝火 | "有话不能说，我感到窝火。" |
| 气愤 | "听到不实的指责，我感到气愤。" |
| 忿忿不平 | "明明是他做错了，却让我承担后果，我忿忿不平。" |

## 重要提醒：区分感受与伪装的想法

在 NVC 中，有一类表达看似在说感受，实际上是对他人行为的评判或对自身处境的判断。这些"伪装的感受"以"我觉得……"开头，但真正表达的是想法而非感受。

**伪装感受的常见模式：**

| 伪装感受（实为想法） | 真实感受 | 原因 |
|---------------------|---------|------|
| ❌ "我觉得被忽视了。" | ✅ "我感到失落和孤独。" | "被忽视"是对他人行为的判断 |
| ❌ "我觉得不被尊重。" | ✅ "我感到愤怒和委屈。" | "不被尊重"包含了对对方的评价 |
| ❌ "我觉得被抛弃了。" | ✅ "我感到害怕和悲伤。" | "被抛弃"是对情境的解读 |
| ❌ "我觉得被利用了。" | ✅ "我感到愤怒和失望。" | "被利用"暗含指责 |
| ❌ "我觉得被误解了。" | ✅ "我感到沮丧和无助。" | "被误解"是对沟通结果的判断 |
| ❌ "我觉得没有被重视。" | ✅ "我感到失落和难过。" | "没有被重视"隐含了对对方的期待 |

**判断技巧：** 如果"我觉得……"后面的内容可以加上"被你"两个字（如"我觉得被你忽视了"），那么这大概率是一个想法/判断，而非纯粹的感受。

## 实践要点

- **感受是信号，不是问题**：负面感受本身不是坏事，它是一个信号，提示你有某个需要没有被满足。
- **先觉察再表达**：感到强烈情绪时，先暂停，觉察身体反应（心跳加速、肌肉紧张、胃部不适），再用词汇描述。
- **避免"你让我……"句式**："你让我很生气"把责任推给了对方。NVC 的表达是："当你那样做时，我感到生气，因为我看重……"
- **接纳所有感受**：没有"不应该有的感受"。允许自己感受愤怒、悲伤、恐惧，是走向自我理解的第一步。

## 常见误区

**误区一：用想法替代感受**

❌ "我觉得你不在乎我。"（这是对对方的判断）
✅ "当你没有回复我的消息时，我感到不安和失落。"（这是感受+观察）

**误区二：把强烈情绪当作全部**

❌ "我简直气炸了！"（虽然真实，但停留在爆发层面）
✅ "我感到非常愤怒，同时也有失望和委屈。"（更完整的觉察）

**误区三：压抑或否认负面感受**

❌ "没什么，我没事。"（否认真实感受）
✅ "我现在有点难过，需要一些时间整理一下。"（诚实面对感受）

**误区四：停留在指责而非探索需要**

❌ "都是你的错，让我这么难受！"
✅ "我现在很难受，因为我很看重信任和沟通。"
