# 正面感受词汇库

## 概述

在非暴力沟通中，清晰地表达正面感受有助于增进人际连接、强化积极体验。许多人在表达正面感受时词汇匮乏，只会说"挺好的"或"不错"。本文档收录了 50 余个正面感受词汇，按四大类别组织，帮助你更精准地表达内心的积极情感。

---

## 一、满足类感受

这类感受源于需要被满足时的内心状态，表达的是一种"足够"和"踏实"。

| 词汇 | NVC 表达示例 |
|------|-------------|
| 满意 | "看到项目顺利完成，我感到很满意，因为我的努力得到了回报。" |
| 充实 | "今天读完了一整章书，我觉得很充实。" |
| 安心 | "知道孩子已经安全到家，我感到安心。" |
| 舒适 | "坐在阳光下喝一杯热茶，我觉得很舒适。" |
| 放松 | "周末不用赶 deadline，我感到特别放松。" |
| 知足 | "一家人围在一起吃饭，我感到很知足。" |
| 坦然 | "把心里话都说出来之后，我觉得坦然了。" |
| 踏实 | "把所有准备工作都做好了，我心里很踏实。" |
| 惬意 | "午后在阳台上晒太阳，我感到十分惬意。" |
| 释然 | "误会解开后，我感到一阵释然。" |
| 心安 | "对方接受了道歉，我终于感到心安。" |
| 自在 | "和老朋友在一起，我觉得很自在。" |
| 宁静 | "独处时听着轻音乐，我感到内心宁静。" |
| 祥和 | "全家人一起散步，气氛很祥和。" |

## 实践要点

- **区分感受和想法**："我觉得自己很优秀"是想法；"我感到自信"是感受。NVC 鼓励表达感受而非评判。
- **从身体信号入手**：正面感受常伴随身体放松、呼吸顺畅、面部微笑等信号，可以通过身体感受来识别情绪。
- **具体化表达**：与其说"我感觉很好"，不如说"我感到满足和安心"——越具体，沟通效果越好。
- **日常练习**：每天睡前回忆三件让你产生正面感受的事，用本文档中的词汇描述它们。坚持练习可以显著提升情绪觉察力。
- **在关系中使用**：当对方做了让你开心的事，直接表达感受比笼统说"谢谢"更有力量——"你记得我的生日，我感到被在乎，这让我很幸福。"

## 常见误区

**误区一：把想法当成感受**

❌ "我觉得自己做得很好。"（这是对自我的评价/想法）
✅ "我对自己的表现感到满意。"（这是真实的感受）

**误区二：用模糊词汇概括一切**

❌ "感觉挺好的。"
✅ "我感到既安心又充实。"

**误区三：忽略正面感受的表达**

很多人习惯只在负面情绪时才开口表达，却忘了正面感受同样需要被表达。及时分享你的喜悦、感激和满足，能大大增进关系中的信任和亲密。

✅ "你主动帮忙洗碗，我感到被支持，心里很温暖。"
